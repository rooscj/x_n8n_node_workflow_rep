X Automation Node for n8n

Overview

This project integrates X (formerly Twitter) with n8n, an open-source workflow automation tool. It enables users to automate interactions with X, such as posting tweets, retrieving user data, and monitoring activity, all within customizable workflows.

Features

  • Post tweets automatically from n8n workflows.
  • Retrieve user timelines and mentions.
  • Monitor hashtags or keywords.
  • Trigger workflows based on X activity.

Authentication

This app uses OAuth 2.0 to authenticate with X. Users must authorize the app to access their account with the required permissions.
